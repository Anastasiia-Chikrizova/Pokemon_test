## Description

```
Web applications that use the PokeAPI are a great way to access and display information about the world of Pokemon. 
They can provide a comprehensive list of all the Pokemon in existence, along with their unique details such as their abilities, stats, and evolutionary paths. 
```

## Load project
```
npm install - install dependencies
npm run start:dev  - load front end part in a dev mode
```


## Scripts
```
- `npm run start` - Launches the frontend project on webpack dev server
- `npm run start:dev` -  Launches the frontend project on webpack dev server 
- `npm run build:prod` - Builds the project in production mode
- `npm run build:dev` - Builds the project in dev mode (not minimized)
- `npm run lint:ts` - Checks ts files using a linter
- `npm run lint:ts:fix` - Fixes ts files using a linter
- `npm run lint:scss` - Checks scss files using a style linter
- `npm run lint:scss:fix` - Fixes scss files using a style linter
```
